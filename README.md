# luc-project
LUC is a structured academic talent platform designed to help universities organize, track, and showcase student skills, projects, and documented learning progress.

This repository is the main monorepo containing both frontend and backend systems.
Repository Structure
apps/frontend – Client application (UI, routing, user experience)
apps/backend – API, business logic, authentication, data handling
packages/shared – Shared types, utilities, validation schemas (if applicable)
docs – Architecture notes and planning documents
